tukituki is a terminal UI and headless CLI for running multiple development processes from a single place. Define your processes once in `.run/*.yaml`, then start them all, watch their logs, and stop them individually — without juggling multiple terminal tabs.

Use the sections below to get up and running quickly or to explore specific features in depth.
